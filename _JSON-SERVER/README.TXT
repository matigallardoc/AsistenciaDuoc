1. Para instalar json-server:

npm install -g json-server

2. Para ejecutar json-server:

json-server --watch posts.json
//o puede ser
json-server --watch _JSON-SERVER/posts.json
//o puede ser
json-server --watch _JSON-SERVER/posts.json --host 0.0.0.0 --port 4000
//o puede ser
json-server --watch _JSON-SERVER/posts.json --host 192.168.4.26 --port 4000
